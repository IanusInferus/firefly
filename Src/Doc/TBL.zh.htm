TBL文件格式表

地狱门神（F.R.C.）制作

本格式表示一个编码映射表，即从一种自定义编码到其实际表示的Unicode形式的变换。

本格式支持可变编码长度，但均必须以字节为单位。因此，如UTF-7等编码无法描述。

本格式支持的最长编码长度没有限制，但由于实现方面的原因，建议不要超过8个字节。

本格式支持的Unicode形式的长度没有限制，但请谨慎考虑，因为较长的Unicode形式会影响编码器性能。

本格式为纯文本格式，编码要求为UTF-8、UTF-16L、UTF-16B、UTF-32L、UTF-32B、GB18030中任意一种。

本格式的语法定义如下：

TBL ::= (Definition | Comment)*

Definition ::= Left "=" Right \r? \n

Left ::= ([0-9A-Fa-f]{2})+

Right ::= EscapedString

EscapedString ::= EscapedUTF16Char+ -> GetUTF16StringAsUTF32String(EscapedUTF16Char+)

EscapedChar ::= "\0" -> U+0000

        | "\a" -> U+0007

        | "\b" -> U+0008

        | "\t" -> U+0009

        | "\n" -> U+000A

        | "\v" -> U+000B

        | "\f" -> U+000C

        | "\r" -> U+000D

        | "\u" UTF16Code:[0-9A-Fa-f]{4} -> GetUTF16CharAsUTF16String(UTF16Code)

        | "\U" UTF32Code:[0-9A-Fa-f]{5} -> GetUTF32CharAsUTF16String(UTF32Code)

        | UTF16Char:. -> UTF16Char

Comment ::= [^=\r\n]*

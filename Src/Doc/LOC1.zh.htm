使用 Variable-Length 编码的字符索引

　

　

注意:

1、所有的整数数据类型都是little-endian的。

2、每个段是16字节对齐的。

3、Variable-Length 编码：对每个32位整数，每7位一段，如果最高有数个段为全0，则这些段省去。对不能省去的段，

   从低位段开始，除最后一个段，每段最高补一位1，构成一个字节，最后一个段最高补一位0。

4、为了便于网络传输，可能使用 RFC 1952 （GZIP 文件格式规范 4.3） 对整个文件进行压缩。但也可以不使用。

   使用压缩时应对整个文件进行压缩，并在起始位置增加标识符"LOCC"。

   .Net Framework 2.0 的 System.IO.Compression.GZipStream 类实现了此压缩。
